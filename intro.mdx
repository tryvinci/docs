# Vinci AI Studio API

Welcome to the Vinci AI Studio API documentation. Our API enables you to integrate powerful AI-driven video and audio manipulation capabilities into your applications.

## Authentication

All API requests require an `X-User-ID` header for user identification.

:::note
Replace `{user-id}` with your actual User ID in all API requests.
:::

## Base URL

```
https://api.tryvinci.com
```

## Available Endpoints

- [Speech to Text](/api-reference/stt) - Convert speech to text
- [Text Translation](/api-reference/translate) - Translate text between languages
- [Text to Speech](/api-reference/tts) - Convert text to natural speech
- [Voice Conversion](/api-reference/voice) - Transform voice to match a target voice
- [Lip Sync](/api-reference/lipsync) - Synchronize avatar lips with audio
- [Live Portrait](/api-reference/live-portrait) - Generate talking head animations
